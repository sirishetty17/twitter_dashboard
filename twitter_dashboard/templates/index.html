Twitter Analytics Dashboard

Analysis of Twitter Bot Usage for Indian Political Issues

This application analyzes Twitter datasets to identify bot accounts, perform sentiment analysis, and present insights through interactive charts and dashboards. It helps understand online political discussions using data-driven analysis.
